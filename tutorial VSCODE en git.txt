1. Lo primero que debemos crear es una variable de entorno para nuestro editor, para ello sigamos los siguientes pasos:
- Debes tener en cuenta el directorio donde esta instalado tu Visual Studio Code, para ello puedes copiarla del asistente de instalación cuando estas en el proceso de instalación o si ya has instalado el aplicativo en tu PC, puedes validar la ruta dándole Click derecho al icono de acceso directo del Visual Studio Code y luego Click en la opción "Abrir la ubicación del archivo". Esto te abrirá una ventana la cual te mostrará la ruta completa donde esta el "Code.exe", en la barra de dirección, copiarla para seguir con el siguiente paso.

- Abrir el "Explorador de archivos" y seleccionar en el menú de la izquierda la opción "Este equipo", en un espacio vacío cerca del Disco C (ojo no encima de algún icono) hacer Click derecho para desplegar el menú emergente y seleccionar la opción "Propiedades", esto te llevara a la ventana "Sistema", al lado izquierdo de esta ventana se mostraran una serie de opciones, escoger la denominada "Configuración avanzada del sistemas", tan pronto hagas Click se abrira una nueva ventana llamada "Propiedades del sistema" con una serie de pestañas, en la pestaña "Opciones avanzadas" se puede observar al final cerca de los botones "Aceptar" y "Cancelar" el botón "Variables de entorno...", hacer Click sobre este.

- Ya dentro de la ventana "Variables de entorno" vamos a ver dos recuadros uno llamado "Variables de usuario para x" (Donde x es el usuario del equipo) y otra que es la que debemos usar llamada "Variables del sistemas", dentro de esta ultima hacer Click en el botón "Nueva...", esto abrira una nueva ventana llamada "Nueva variable del sistema". En "Nombre de la" vamos a darle el nombre que queramos darle a nuestra variable del sistema asociada al Visual Studio Code, esta puede ser personalizada sin embargo se recomienda no dejar espacios, para este caso usaremos "VSCode", luego en el campo "Valor de la" pegaremos la ruta que hemos copiado del directorio donde esta nuestro Visual Studio Code, en mi caso C:\Users\jorge\AppData\Local\Programs\Microsoft VS Code, tener en cuenta que debemos finalizar la ruta con \Code.exe, en este punto deberia quedarnos: C:\Users\jorge\AppData\Local\Programs\Microsoft VS Code\Code.exe, luego hacemos Click en "Aceptar", "Aceptar" y "Aceptar".

- Una vez creada nuestra variable de entorno podemos agregar el editor Visual Studio Code a Git, para ello abrimos el Git Bash en nuestra consola de Git escribimos: git config --global core.editor "VSCode --wait" y damos enter.
